# Habit Tracker

A habit tracking app that helps you build consistency. Track daily habits, see your streaks, and visualize progress with a clean heatmap.

## Features

- Create and manage habits
- Mark daily completions
- Track streaks automatically
- Visual progress tracking with heatmap
- Sort by name, streak length, or current streak
- Delete habits you no longer need
- Works on mobile and desktop

## Stack

**Backend:** Node.js, Express, MongoDB, Mongoose  
**Frontend:** React, Tailwind CSS

## Requirements

- Node.js v14+
- npm or yarn
- MongoDB (Atlas or local)

## Quick Start

### Backend

```bash
cd backend
npm install

# Set up .env file
# MONGODB_URI=<your-mongodb-url>
# PORT=5000
# NODE_ENV=development

npm start
```

Runs on `http://localhost:5000`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on `http://localhost:5173`
